# general

This utility let you generate a readable markdown file from your
`keybindings.xml` file.

# usage

In windows explorer, go to : `C:\Users\your_name\AppData\Renoise\v3.x.x`

Copy the path.

Run the app.

Paste path there, and hit `Save`

You now have a MD file next to your key confs, read it in obsidian or vscode.
